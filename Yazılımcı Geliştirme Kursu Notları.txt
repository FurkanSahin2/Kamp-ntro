Soru1:
C# dilinde "ternary operatörü" nedir?

Cevap1: 
Ternary Operatörü'nü if-else bloğunun kısayolu olarak düşünebiliriz.

Normal bir if-else buloğu şu şekildedir;

if(Kontrol edilecek KOŞUL)
{
     //Koşul sağlanıyorsa ise çalıştıracağımız kodları bu bloklar arasına yazarız.
}
else
{
     //Koşul sağlanmıyorsa çalıştıracağımız kodları bu bloklar arasına yazarız.
}

Ternary Operatöründe ise şu şekildedir;

(Kontrol edilecek koşul)?  "Koşul sağlanıyorsa " : "Koşul sağlanmıyorsa"

Soru2:
C# dilinde "switch" yapısı nedir?

Cevap2:
Switch-case yapısında belirlediğimiz verinin değerini kontrol ediyoruz.Örneğin

Switch(Verimiz)
{
     case değer1:
          Değer doğruysa çalışacak kodları yazıyoruz.
          break;
     case değer2:
          Değer doğruysa çalışacak kodları yazıyoruz.
          break;
     default:
          Değer sağlanmazsa çalışacak kodları yazıyoruz.
          break;
}

herhangi bir case'in içindeki kod çalıştıktan sonra işlemi durdurmak için "break" komutunu kullanıyoruz.
default ise değerlerimiz sağlanmaz ise çalışacak kod'u belirliyor if-else yapımızdaki else benzetebiliriz.

Soru3:
C# dilinde "while döngüsü" nedir?

Cevap3:
while döngüsü belirlediğimiz koşul sağlandığı süre boyunca kodumuzu tekrar tekrar çalıştırmamızı sağlar. Örneğin:

int sayi = 0;
while(sayi<100)
{
     Console.WriteLine("Muhammed");
     sayi++;
}

sayi++ demek döngü her çalıştığında sayi'yi 1 arttır demektir. Yani döngü her çalıştığında sayi 1,2,3,4,5,6... 
diyerek koşulumuzda belirttiğimiz gibi sayi 100'den küçük olmayıncaya kadar çalıştıracak ve ekrana 100 defa adımı yazmış olacağım.
